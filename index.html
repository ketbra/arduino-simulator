Arduino Simulator Help

Building Circuits

  • Add components: Click buttons in the palette (LED, RGB LED, etc.)
  • Connect wires: Click a pin, then click another pin to connect them
  • Cancel wire: Press Esc or click empty space
  • Move components: Drag them — wires follow automatically
  • Delete: Click to select, then press Delete or right-click → Delete
  • Push button: Click the round cap to press/release

Running Code

  • Run: Executes your code continuously
  • Step: Execute one line at a time (click Next to advance)
  • Stop: Halt execution
  • Distance slider: Controls the virtual ultrasonic sensor

Keyboard Shortcuts

  • Ctrl+Z — Undo
  • Ctrl+Y / Ctrl+Shift+Z — Redo
  • Delete / Backspace — Delete selected component or wire
  • Esc — Cancel wire in progress
